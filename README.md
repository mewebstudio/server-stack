# Server stack for Docker Swarm

### Requirements:
* Docker with Swarm mode

### Applications:
* NGINX Proxy Manager
* Portainer
* Jenkins
* PostgreSQL
* PgAdmin
* MariaDB
* PhpMyAdmin
* Adminer
* MongoDB
* mongo-express
* Redis
* Redis Commander
* RabbitMQ
* Elastic Search
* Kibana

----------------------------------------------------------------

### Install and run with Docker
```bash
docker-compose up --build -d
````
